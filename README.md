■ アプリ概要
本アプリは、数字を極力排除し、円形のゲージの増減によって視覚的に時間を管理するタイマーです。
作業時間と休憩時間を自動で循環させる機能を備えています。

■ 動作環境
本アプリはWebベースで動作します。以下の環境での利用を推奨します。

・OS: 
    - Android 10.0以上 (Chromeブラウザ推奨)
    - iOS 15.0以上 (Safariブラウザ推奨)
    - Windows 10/11 (Chrome, Edge, Firefox)
    - macOS (Chrome, Safari, Firefox)

・ブラウザ:
    - Google Chrome (最新版)
    - Safari (最新版)
    - Microsoft Edge (最新版)

※ JavaScriptが有効な環境である必要があります。

■ 必要容量・データ通信について
・必要容量:
    - 本アプリは40KBと非常に軽量なWebアプリケーションです。インストール等は不要（ブラウザで動作）ですが、キャッシュとして数MB程度の領域を使用する場合があります。
・データ通信:
    - 初回読み込み時に必要なファイルをダウンロードしますが、それ以降のタイマー動作自体はオフラインでも利用可能です。

■ 主な機能
・作業時間・休憩時間のカスタマイズ（1分単位）
・ゲージカラーの自由選択
・ダークモード対応
・アラームの回数・種類の選択（8種類）
・テスト再生機能（アラーム音の確認用）
・長押しリセット機能
・バイブレーション通知機能（iOSはセキュリティ対策のため不可能）

■ インストール方法（ホーム画面への追加）
本アプリをお使いの端末にインストールするには、以下の手順でホーム画面に追加してください。

 [iOSの場合]
  1. Safariで本アプリのURLを開く
  2. 画面右下の「･･･」メニューをタップ
  3. 「共有」を選択後、「表示を増やす」をタップ
  4. 「ホーム画面に追加」を選択
  5. 「Webアプリとして開く」にチェックを入れて「追加」をタップ
[Androidの場合]
  1. Chromeで本アプリのURLを開く
  2. 画面右上の「︙」メニューをタップ
  3. 「ホーム画面に追加」を選択
  4. 「インストール」を選択

■ 操作方法
・タップ: タイマーの開始 / 一時停止
・長押し: タイマーのリセット
・右上アイコン: 各種設定の変更
